Airlines Safety Dataset and Code Text


This project showcase the number of incidents, fatal accidents and fatalities that occurred within major airline Companies across the world between 1985 to 2019 (YTD - June 2019). The purpose for this write-up is to show at-risk airline Companies and help customers make an informed decision when choosing an airline to fulfill their traveling needs. However, it also shows that as compared to years 1985-1999, 2000-2014 and 2015-2019 (YTD), great progress has been made by most of these airlines to reduce the number of fatalities/deadly accidents across the world.

The attached zip file contains the dataset (CSV) and the ipynb code text used in building the report. To access the dataset, open the zip folder and click on the csv file. For the purpose of this presentation, only about 25 records were populated with Lat and Long. for the showcase. Feel free to add more according to your needs. The Notebook contains the code used to build our dashboard with step by step explanation.
